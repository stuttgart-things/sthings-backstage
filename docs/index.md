# sthings-backstage

Welcome to the documentation for the Stuttgart Things Backstage Internal Developer Portal.

## Overview

This Backstage instance serves as the central developer portal for Stuttgart Things, providing:

- **Software Catalog**: Centralized service and component registry
- **TechDocs**: Technical documentation as code
- **Software Templates**: Scaffolding for new projects
- **Kubernetes Integration**: Cluster visibility and management
- **Search**: Unified search across all entities
